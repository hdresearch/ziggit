=== KEY INSIGHTS ===

1. PURE ZIG ADVANTAGE: Direct Zig function calls eliminate process spawn overhead (1ms+ saved per call)
2. FILE I/O DOMINATES: status operations are limited by file system access patterns, not computation
3. OPTIMIZATION TARGETS: status --porcelain and is_clean can be further optimized with:
   - Better index caching
   - Ultra-fast clean path improvements  
   - Reduced file system calls
